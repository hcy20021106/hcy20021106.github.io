MachineLearning

Posted on 2025-06-16 Edited on 2025-06-17 In AI

Basic Concept

Look for a function for e.g., image recognition, speech recognition.

  • Regression
    The function outputs a scalar
  • Classification
    Given options (classes), the function outputs the correct one.
  • Structured Learning

Linear Model

Define function with unknown parameters

Model: $y = b + w_1$ based on domain knowledge
w (weight) and b (bias) are unknown parameters (learn from data)
如果是一组包含有多个feature（例如同时考虑前七天的数据来进行预测）
那么$y = b + \sum_j w_jx_j$

Define Loss from Training Data

Loss is a function of the unknown parameters: L(b,w)
e.g., L(0.5k, 1) corresponding to $y = 0.5k + 1x_1$
我们定义一个loss function为:
$e_n = |y_n - \hat{y_n}|$

Loss: $L(b, w) = \frac{1}{N} \sum_{n=1}^{N} \left| y_n - \hat{y}_n \right|$

其中y为预测值，$\hat{y}$为真实值,这里我们定义的Loss是mean absolute error （MAE），如果是mean square error（MSE），则是差值的的平方。

Optimization

我们通过最小化损失函数 L(b, w) 来学习未知参数 b 和 w。优化目标为：

$b^, w^ = \arg\min_{b, w} L(b, w)$

我们可以使用梯度下降法进行优化，参数更新如下：

$w \leftarrow w - \eta \frac{\partial L}{\partial w}, \quad
b \leftarrow b - \eta \frac{\partial L}{\partial b}$

其中 $\eta$ 为学习率。对于 MAE 损失，由于其不可导点较多，可以使用次梯度法；对于 MSE 损失，则可以直接使用标准梯度下降算法。

Sophisticated Models

Beacuse linear models have severe limitation (model bias), so we include more sophisticated models.

Sigmoid function

$y = b + wx_1 => y = b + \sum_i c_i \cdot \sigma(b_i + w_ix_1)$
注：不同的w，b，c可以让sigmoid曲线可以拟合不同的曲线
注意这里是多个sigmoid的和，因为实际情况可能是由多个sigmoid拟合而成。
相对应的，对于$y = b + \sum_j w_jx_j$, 则有$y = b + \sum_i c_i \cdot \sigma(b_i + \sum_j w_{ij} x_j)$

示例：i = 1,2,3; j = 1,2,3 时的显式形式
$$
r_1 = b_1 + w_{11}x_1 + w_{12}x_2 + w_{13}x_3
$$

$$
r_2 = b_2 + w_{21}x_1 + w_{22}x_2 + w_{23}x_3
$$

$$
r_3 = b_3 + w_{31}x_1 + w_{32}x_2 + w_{33}x_3
$$

向量化（矩阵）形式
$$
\mathbf{r} = W \mathbf{x} + \mathbf{b}
$$

然后对生成的r做sigmoid函数，矩阵表达形式如下

$$
\mathbf{a} = \sigma(\mathbf{r})
$$

最后的输出y矩阵表达形式如下：
$$
y = b + \mathbf{c}^T \mathbf{a}
$$

Optimization部分，先将所有unknown parameters放到$\mathbf{\theta}$向量中
$$
\mathbf{\theta} = \left[ b,\ c_1,\ \dots,\ c_i,\ b_1,\ \dots,\ b_i,\ w_{11},\ \dots,\ w_{ij} \right]
$$

定义Loss function $L(\theta)$,最后想要取得的 $\mathbf{\theta^{*}} = \arg\min_{\theta} L$

因此我们同样的使用梯度下降更新去得到最后的$\mathbf{\theta^{*}}$
$$
\theta \leftarrow \theta - \eta \cdot \nabla_\theta L(\theta)
$$

Batch Size, Epoch, N
1,000 examples (N = 1,000), Batch size is 100, so 100 update in 1 epoch (10 Batch).

信源编码

Posted on 2025-06-07 In 通信原理

Basic Concept

信源编码主要用于提高数据传输的可靠性，纠正通信过程中可能发生的错误。常用的信源编码包括：

  • Reed-Solomon（RS）编码：一种块编码，擅长纠正突发错误。
  • 卷积编码：一种常见的前向纠错码，适用于连续数据流。
  • 交织（Interleaving）：重排数据顺序，扩散突发错误，提高纠错效果。

整体收发流程为:

  1. 发射端
    视频源-》RS编码-》卷积交织-》数据流标记-》
  2. 接收端
    解调后的基带数字信号-》维特比解码-》解交织-》RS解码

Reed-Solomon (RS) 编码

RS编码基本概念

  • RS编码在数据块后添加冗余校验码，能纠正一定数量的错误字节。
  • 典型参数：RS(n, k)，n为编码后数据长度，k为原始数据长度。
  • 纠错能力为 t = (n - k) / 2。

基本概念

  • 把相邻字节分散到不连续的位置
  • 把突发错误转换为随机错误，提高纠错性能

正交调制

已调信号表示为正交调制信号的组合
$u(t)=A(t)cos[w_ct+\phi(t)] = I(t)cosw_ct-Q(t)sinw_ct$

其中$I(t)=A(t)cos\phi(t)$为同相分量
$Q(t)=A(t)sin\phi(t)$为正交分量
易见I(t)和Q(t)都是成形滤波，因此在生成I(t)和Q(t)我们先引入In和Qn，可以在信号矢量图/星座图上面显示In，Qn
$u_n(t-nT_B) = A_ng(t-nT_B)cos[w_ct+\phi_n] = I_ng(t-nT_B)cosw_ct-Q_ng(t-nT_B)sinw_ct$
注意u(t)和$u_n(t)$的区别在于是否对单个码元进行分析，$I_n$和$I_t$的区别也是如此。
其中对$I_ng(t-nT_B)$和$Q_ng(t-nT_B)$求累加可以分别表示I(t)和Q(t)，因此In和Qn可以表征It和Qt

正交调制的相干解调法

由于$u(t) = I(t)cosw_ct-Q(t)sinw_ct$

  • 获取In
    与本地载波$cosw_ct$相乘后经过低通滤波器后生成$\frac{1}{2}I(t)$,然后经过抽样判决后生成In
  • 获取Qn
    与本地载波$-sinw_ct$相乘同理获取Qn

然后In与Qn经过并/串变换后得到输出01序列

模拟调制

Posted on 2025-06-04 Edited on 2025-06-16 In 通信原理

AM（Amplitude Modulation）调幅

基本概念

  • 通过改变载波信号的幅度来携带信息信号。
  • 调制信号形式：

$$
s(t) = [A_0 + m(t)] \cos(2\pi f_c t)
$$

其中$A_0\cos(2\pi f_c t)$为载波，$m(t)$为调制信号，$s(t)$为已调信号

  • 简单易实现。
  • 带宽为 (2B)，其中 (B) 为信号带宽。
  • 存在包络检波方式，易于接收。
  • 效率低，载波本身不包含信息但占用功率。

包络检波器包括全波整流器和低通滤波器两部分组成，其中全波整流器用于将负半轴的波形翻到正半轴

功率计算

$S_{AM}(t) = [A_0 + m(t)]cosw_ct = A_0cosw_ct + m(t)cosw_ct$
其中$A_0cosw_ct$和$m(t)cosw_ct$分别是载波相和边带相，其功率分别是$\frac{A_0^2}{2}$和$\frac{\overline{m(t)}^2}{2}$

Notice

$\overline{m(t)}^2$表示m(t)的均方值
对随机过程或者给定信号都能求均方值，并且其都用来求平均功率

DSB（Double Side Band）双边带调制

由于包络与m(t)不成正比，不能使用包络检波法，此时解调需要使用相干解调法

  • 又称 DSB-SC（Suppressed Carrier）。
  • 将基带信号与载波相乘，抑制直流项。
  • 形式：

$s(t) = m(t) \cos(2\pi f_c t)$

特点:

  • 没有载波，传输效率高于 AM。
  • 但接收时需要载波同步，复杂度增加。
  • 频谱包含上下两个边带，带宽为 (2B)。

SSB（Single Side Band）单边带调制

以调制信号为单频正弦波$m(t)=A_mcosw_mt$为例
$s_{DSB}(t) = A_m cosw_mtcosw_ct = \frac{1}{2}A_mcos(w_c+w_m)t + \frac{1}{2}A_mcos(w_c-w_m)t$

其中上边带$s_{USB}(t) = \frac{1}{2}A_mcos(w_c+w_m)t$
下边带$s_{LSB}(t) = \frac{1}{2}A_mcos(w_c-w_m)t$

基本概念

  • 在 DSB的基础上，仅保留一个边带（上边带或下边带）。
  • 去除冗余频谱部分，提高频谱效率。

生成方法:

  • 滤波法：滤除一个边带。
  • 平衡调制 + 相移法（Hilbert 变换）：生成正交信号组合消除一个边带。

特点:

  • 带宽减半，仅为 (B)。
  • 频谱效率高，适合长距离传输（如语音通信）。
  • 实现复杂，载波恢复要求高。

Notice

Hilbert滤波器使得信号在时域上偏移$\pi/2$
$A_mcosw_mt$经过Hibert变换后得到$A_mcos(w_mt-\pi/2)$，也就是$A_msinw_mt$
因此Hilbert滤波器的冲击响应$h_h(t) = \frac{1}{\pi t}$,其传递函数$H_h(w)=-jsgn(w)$，
其中$\operatorname{sgn}(\omega) =\begin{cases}
+1, & \omega > 0 \
0, & \omega = 0 \
-1, & \omega < 0
\end{cases}$

VSB（Vestigial Side Band）残留边带调制

  • 在 SSB 基础上，保留一个完整边带 + 另一个边带的一小部分（残留边带）。
  • 实际用于电视视频信号等高带宽场景。

特点:

  • 兼顾频谱利用与实现难度。
  • 能较好保留低频成分，解决 SSB 滤波困难问题。
  • 广泛用于模拟电视广播（如 NTSC、PAL）。

FM/PM（角度调制）

$S_m(t) = Acos[w_ct+\phi(t)]$
其中：
顺时位移=$w_ct+\phi(t)$
瞬时角频率=$w_c+\frac{d\phi(t)}{dt}$
PM调相是让其中的$\phi(t)$与调制信号成正比，也就是$\phi(t)=k_pm(t)$
FM调频是让其中的$\frac{d\phi(t)}{dt}$与调制信号成正比，也就是$\frac{d\phi(t)}{dt}=k_fm(t)$
实际应用中通常使用FM调频

FM

令$m(t)=A_mcos(w_mt)$
这个调制信号也是单因信号
根据$\frac{d\phi(t)}{dt}=k_fm(t)$可知
$\phi(t) = K_fA_m\int cos(w_m\tau)d\tau$ = $\frac{K_fA_m}{w_m}sin(w_mt)$
其中$m_f$为调频指数

NBFM

FM信号的最大瞬时相位偏移满足远小于pi/6(0.5)
通过化简可得
$S_{NBFM}(t) = Acos(w_ct)-[AK_f\int m(\tau)d\tau]sin(w_ct)$

Notice

对于直流分量为0的信号，也就是其傅立叶变换后的w=0处为0的信号，有
$\mathcal{F}\left(\int m(t),dt \right) = \frac{M(\omega)}{j\omega}$

hint：
不定积分可以写成原信号和单位阶跃函数卷积的形式，然后单位阶跃信号的傅立叶变换为 $\frac{1}{jw} + \pi\delta(w)$
由于原信号的傅立叶变换后的w=0处为0，因此与冲击信号乘积的部分可除去。

FM信号的解调

$s_{FM}(t)$经过在接收端经过带通滤波器及限幅后再经过微分电路然后再经过包络检波以及低通滤波器
$s_{FM}(t)$在经过微分后生成的$s_d(t) = -A[w_c+K_fm(t)]sin[w_ct+K_f\int m(\tau)d\tau]$

然后经过包络检波后可以得到$-A[w_c + K_fm(t)]$，最后经过低通滤波器滤去wc从而得到m(t)

模拟调制系统的抗噪声性能

相干解调法

研究抗噪声性能也就是在研究SNR，为了研究模拟调制对系统的抗噪声能力，我们重点关注已调信号$s_m(t)$与n(t)在经过带通滤波器（输入端的接收滤波器）后生成的$s_m(t)$与$n_i(t)$的$\frac{S_0}{N_0}$与经过解调器后形成的$\frac{S_i}{N_i}$之间的比值。
结论：
以DSB为例，最后的信噪比经过相干解调后会改善一倍，也就是$G_{DSB} = \frac{S_0/N_0}{S_i/N_i}=2$
这是因为在相干解调的过程中噪声的正交分量被低通滤波器滤除了

Notice

实质上已调信号$s_m(t)$会经过发送滤波器和信道，但我们在这里将其忽略，因此只需考虑$s_m(t)$
另外，我们假设$s_m(t)$经过带通滤波器后的保持不变，仍为$s_m(t)$
而对于噪声n(t) (我们通常假设其为高斯白噪声)，在经过带通滤波器的传信滤噪后，会生成窄带噪声，因此我们用窄带噪声的特性来分析

调制方式 解调前信噪比 $\frac{S_0}{N_0}$ 解调后信噪比 $\frac{S_i}{N_i}$ 抗噪声能力 $G = \frac{S_i/N_i}{S_0/N_0}$ 特点说明
AM（含载波） $\frac{\frac{A_0^2}{2}+\frac{\overline{m^2(t)}}{2}}{n_0B(N_i)}$ $\frac{1}{4}\overline{m^2(t)}/ \frac{1}{4}N_i(N_0)$ $\frac{2\overline{m^2(t)}}{A_0^2+\overline{m^2(t)}}$ 信噪比依赖调幅指数，抗噪性能最弱
DSB（无载波） $\frac{1}{2} \overline{m^2(t)} / n_0B(N_i)$ $\frac{1}{4}\overline{m^2(t)} / \frac{1}{4}N_i(N_0)$ 2 相干解调提升一倍信噪比
SSB $\frac{1}{4}\overline{m^2(t)} / n_0B(N_i)$ $\frac{1}{16}overline{m^2(t)} / \frac{1}{4}N_i(N_0)$ 1 最佳带宽效率，抗噪不变

包络检测法

AM系统

非线性解调，不能分别处理信号和噪声
输入信号$S_{AM}(t) = [A_0 + m(t)]cosw_ct$
由于输入会经过带通滤波器，因此输入噪声会成为窄带高斯白噪声，根据窄带高斯白噪声的性质可将其分为正交部分和同相部分两部分，如下输入噪声所示：
输入噪声$n_i(t) = n_c(t)cosw_ct - n_s(t)sinw_ct$

将二者信号相加并通过包络检测可得到
所含包络$E(t) = \sqrt{ \left[ A_0 + m(t) + n_c(t) \right]^2 + n_s^2(t) }$
后面需要分析大信噪比和小信噪比两种情况

大信噪比

信号包络远超过噪声包络
也就是$A_0+m(t)>>\sqrt{n_c(t)^2 + n_s(t)^2}$
因此$E(t) \approx \sqrt{[A_0+m(t)]^2 + 2[A_0 + m(t)]n_c(t)} \approx A_0 + m(t) + n_c(t) $

Notice

$(1+x)^{1/2} \approx 1 + \frac{x}{2}$

因此输出信号中信号(m(t))的平均功率为$\overline{m(t)^2}$, 噪声的平均功率为$\overline{n_c(t)^2}$
其中$n_c(t)$为窄带高斯噪声，功率为其均方值，也就是$E[n_c(t)^2]$,根据窄带信号的统计特性可知，其本身的数学期望和方差都和其正交部分和同相部分相等，因此均方值也相同
因此此时输出信噪比$S_0/N_0$与相干检波时一样，又因为输入信噪比也一样（易知），因此抗噪声性一样（大信噪比情况下）

小信噪比

也就是信号包络远小于噪声包络
通过分析合成的信号的包络E(t)，可知信号会被检测器扰乱为噪声。因此此时输出信噪比急剧恶化
这也就是门限效应，是由于包络检测器的非线性解调作用而产生的。（相干解调是线性解调，不存在门限效应）
在大信噪比环境下抗噪声能力与相干解调在任意信噪比时几乎相同，而在小信噪比时会很弱，因此会有一个拐点，也就是会出现门限效应。
