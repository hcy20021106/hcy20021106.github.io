MVC

Posted on 2025-06-12

🧠 MVC Architecture

🟢 Model (PizzaModel.java)

  • Manages pizza size and toppings
  • Maintains pricing data for sizes and toppings
  • Calculates total price dynamically

Size Prices

Size Base Price
Small €4.00
Medium €5.50
Large €7.00

Topping Price (based on selected size)

Size Topping Price
Small €0.75
Medium €1.00
Large €1.45

🟡 View (PizzaView.java)

  • Built with Java Swing
  • Contains:
    • Size selection (radio buttons)
    • Toppings selection (radio buttons)
    • Total price display (JTextField)
    • Pizza image (Pizza.jpeg)

Layout uses BorderLayout with components divided into size panel, topping panel, total price panel, and center image.

🔵 Controller (PizzaController.java)

绑定View与Model

  • Binds user interaction to logic
  • Registers ActionListener to size and topping buttons
    使用SizeRadioButtonListener类实现了ActionListener接口，用来监听用户点击按钮的行为。
    然后给button通过addActionListener这个类从而实现监听功能
  • On user interaction:
    • Updates model with size/toppings
    • Recalculates and updates price in the view

💡 How It Works

  1. User selects a pizza size.
  2. Topping prices are adjusted based on size.
  3. User selects/deselects toppings.
  4. Total price updates live on the GUI.
