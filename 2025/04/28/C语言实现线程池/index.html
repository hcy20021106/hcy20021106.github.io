C语言实现线程池

Posted on 2025-04-28 In C++

Basic Concept

在高并发程序中，频繁创建和销毁线程会造成较大的资源消耗和系统开销。线程池（Thread Pool）是一种常用的并发编程技术，通过维护一个线程集合来处理任务，避免频繁创建和销毁线程，提高系统性能
线程池通常具备以下功能：

  • 支持动态添加任务
  • 线程复用，减少线程创建销毁的开销
  • 任务队列支持FIFO机制
  • 线程池的销毁与资源回收
